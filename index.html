오후 5시에 강의가 끝나도 뜻이 맞는 학과 동기들끼리 남아서 과제학습, 알고리즘 풀이, Spring MVC 등을 밤 10시까지 공부하였습니다.
그 결과 성실성을 인정받아 교수님의 추천으로 조기취업을 했습니다.

프로젝트 실습과목에서 Spring MVC, Mybatis, BootStrap5, Flask, Redis, MongoDB 등을 공부하여 프로젝트 3개를 진행했으며
혼자서 처음부터 끝까지 CRUD, 페이징 처리, 채팅기능, 인증, 세션, 등을 구현할 수 있습니다.

현재는 Spring 핵심원리(DI, IOC, Solid, AOP), Spring JPA Data, Spring Security, JWT 등을 학습하고, 프로젝트를 완료했습니다.
빠르게 기능을 구현하고 잘 모르는 부분은 기술부채로 남겨두었습니다.
이후 기술부채를 공부할 때는 코드로 친숙해진 상태이기에 더욱 재밌게 공부했습니다.

최근에는 실시간 데이터 스트리밍을 구현해보고 싶어서 오픈서치 대시보드로 모니터링 시스템을 구현했습니다.
Spring Boot → WebClient → Kafka → Logstash → OpenSearch 데이터 파이프라인을 구축하였고
실시간으로 웹 앱에서 발생한 네트워크 로그가 OpenSearch Dash board에 업데이트 되도록 구현했습니다.
