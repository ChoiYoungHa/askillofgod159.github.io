저는 지금 이것을 할 수 있습니다.

개발의 처음 시작은 Java로 시작했습니다. 덕분에 객체지향 프로그래밍에서 중요한 캡슐화, 추상화, 상속, 다형성 같은 개념을 깊이 있게 이해할 수 있었습니다. 이러한 기본적인 개념들을 바탕으로 컴퓨터 사이언스(데이터베이스, 운영체제, 네트워크, 자료구조 등)를 공부하고 현재는 Spring 프레임워크 서버 개발자를 준비하고 있습니다. 간단한 웹 서비스를 혼자서 개발할 수 있습니다. 자바 기반의 검색엔진 서버 운영경험은 기술적인 면뿐만 아니라 문제해결력을 키우는데도 큰 도움이 됐습니다. 예를들어, 실시간 문제를 해결하기 위해 데이터 구조와 알고리즘을 적용하는 방법을 배웠고, 이는 대용량 트래픽 환경에서 빠른 응답 시간을 보장할 수 있었습니다. 사용자로부터 실시간 데이터를 받고, 인기 검색어 Top 5를 실시간으로 제공해야하는 상황에서 Hashmap을 사용했습니다. 검색어를 키로, 검색량을 값으로 사용합니다. 알고리즘 적용으로 Top 5 검색어를 추출하기 위해 최소 힙(Min-Heap) 구조를 사용하는 우선순위 큐를 적용할 수 있습니다. 큐는 검색어를 검색량에 따라 자동으로 정렬해주고, 상위 5개의 키워드만 유지하여 새로운 데이터가 추가될 때마다 가장 작은 값을 제거합니다. 이렇게 Java로 알고리즘 공부를하고, 스프링 프로젝트를 했으며, 검색엔진 서버운영도 Java를 사용했습니다.

진행한 프로젝트 배포는 AWS EC2를 통해 배포하였고, Docker 및 Jenkins를 이용하여 CI/CD 환경을 구축하였습니다. 보안 및 동시접속 처리를 대비하여 Nginx를 프록시 서버로 사용하였고 SSL 인증서를 받았습니다. 그 이외에 AWS 인프라 구축 경험은 데이터 엔지니어로 일 했을 때 보안을 위해 Secret Manager를 Boto3 라이브러리로 사용 하였으며, S3에 데이터레이크 구축, MWAA로 트러블 슈팅관리, Glue 대용량 데이터 처리, Lambda 함수처리 등 AWS 기반의 인프라를 관리한 경험이 있습니다.
